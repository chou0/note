## 十进制转换成二进制
* 需要注意 带小数的十进制转换
1. 十进制整数部分转化为二进制
```
假如 x 为 十进制 整数
x = a*2 + b*2*2 + c*2*2*2 + .....
x 二进制表示为   00000cba
```
 算法：用 x 循环除以2，列出余数就是 该数的二进制表示 

2. 十进制小数部分转化为二进制
```
假如 x 为 十进制 小数
x = a*(1/2) + b*(1/2/2) + c*(1/2/2/2) + ....
x 二进制表示为 0.abc
```
 算法：用 x 循环乘以2，列出 整数部分 就是该数的二进制部分

## 二进制转换成十进制
```
假如 x 为二进制
1010.1001
```
* 计算十进制
